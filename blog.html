Discuss the scope of var, let, and const

var is a globally scoped variable. But if you us it on function you only use it on function. variables can be updated and re-declared within its scope.

let const is a block scoped variable. let variables can be updated but not re-declared. const variables can neither be updated nor re-declared.

Tell us the use cases of null and undefined

Undefined means the variable has been declared, but its value has not been assigned. Null means an empty value or a blank value The typeof() operator returns undefined for an undefined variable. The typeof() operator returns the type as an object for a variable whose value is assigned as null.

What do you mean by REST API?

API is application programming interface. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
